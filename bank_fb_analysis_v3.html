06 洞察 Insights

留言抽獎是 FB 最有效的互動武器

台新易匯通文 1,661 ❤ + 1,351 留言 + 890 分享；合庫兒童節文 738 ❤ + 687 留言。「情境痛點 → 解法 → 留言抽獎」三段結構是共同公式，搭配投放可放大 10–50 倍互動。

中信三層閉環最值得複製

FB 貼文 → App 任務牆 → OPENPOINT 抽獎 → 再用 App。每則貼文帶 3 個以上連結，IP 聯名提升分享意願，是導流閉環設計的最佳範例。

玉山「好友日」是少見的定期整合機制

每週固定特定日發布多品牌優惠整合貼文（Uber Eats、小米、證券等），形成固定追蹤預期。不靠單一大活動，靠週期感建立習慣。

合庫偶發性互動潛力被低估

日常貼文互動 18–45 ❤，但兒童節留言抽獎單則爆出 1,802 綜合互動，顯示合庫粉絲有回應能力，只是缺乏機制設計來規律啟動。

稅務季是可搶佔的話題空白

玉山信用卡繳牌照稅貼文拿下 164 ❤，是其樣本最高。彰銀也有進場。但台新、中信、國泰幾乎缺席。4–6 月牌照稅 + 房屋稅是全民話題，空檔明顯。

彰銀「雙周報」是品牌媒體化的孤例

六家中唯一系統性做品牌敘事系列的品牌。互動數雖低，但建立長期追蹤習慣，是不靠投放也能持續有曝光的路徑，適合預算有限的長期策略。

Richart：FB 是輔助，IG 才是主場

Richart 擁有 24.2 萬 FB 粉絲但發文頻率偏低（週 1–2 則），互動數估算在競品中排末段。品牌主力內容放在 IG 的「圓環旁說書人」帳號，FB 僅作為開戶活動與優惠廣播的輔助渠道。這是台新金控有意識的雙帳號分工策略。

理財教育內容是真正的藍海

合庫有做防詐/安養信託教育，玉山有做數位帳戶問答體教學，但兩者都還不夠系統化。其餘四家幾乎沒有固定教育系列。建立週更理財教學欄目，是用內容建立長期信任的最低競爭空間。

FB 右上象限目前無人佔據

「平台留存 + 機制型互動」象限空白。若能設計不把人導出 FB、但有互動誘因的貼文（例如留言完成任務直接獲獎），將形成清楚差異化定位。

07 第一銀行可執行機會點
基於競品分析，以下為第一銀行最具可行性的 4 個切入方向：
立即可做：留言抽獎機制設計

參考台新/合庫公式，情境痛點 + 留言抽獎 + iLEO 導流，一則貼文同時拉互動與開戶。

立即可做：稅務季搶先佈局

玉山已入場，其他競品缺席。4月牌照稅攻略懶人包（台灣Pay + iLEO），搶先卡位話題。

中期建立：好友日式固定優惠整合

參考玉山好友日，建立「小粉獅好康週」固定機制，整合信用卡、外幣、保險等多產品優惠。

長期差異化：理財教育固定欄目

六家競品都未做系統性理財教育。以「獅獅理財小視窗」為基礎建立週更系列，用知識建立長期信任。

08 總結

目前銀行 Facebook 社群經營的主要趨勢

台灣銀行品牌的 FB 策略正從「優惠公告板」演化為「互動型轉換平台」。 核心競爭已不在回饋率高低，而在於誰的互動機制設計得更聰明。 台新靠留言抽獎 + 媒體投放衝爆量，中信靠IP 聯名 + App 閉環深度導流， 玉山靠好友日週期整合建立固定觸及，合庫靠教育內容慢慢累積信任。 彰銀的品牌媒體化是另一條路——不靠投放、靠內容建立長期資產。 短期轉換與長期品牌需要雙線並行，是這個時代銀行社群最重要的策略判斷。
